# Data

This folder contains datasets used for FCIP compliance risk model development.

## Primary Data Source
- FDA Import Refusal Reports (publicly available)
  - Source: https://www.fda.gov/food/compliance-enforcement-food/import-refusal-reports
  - Used for: Training the compliance risk prediction model (Module 1)

## Planned datasets:
- `fda_import_refusals.csv` — FDA historical import refusal records
- `product_categories.csv` — Product category risk classifications
- `compliance_features.csv` — Processed compliance feature dataset for ML model

## Data Notes
All datasets used in this project are sourced from publicly available 
regulatory databases. No proprietary or restricted data is used.
